API Documentation

このAPIを使用すると、以下のデータを取得できます。
・ワインのカテゴリー
・味わい
・生産国
・使用しているぶどう品種
・該当ワインに合う料理
・平均的な価格

エンドポイント

URL: https://sheetdb.io/api/v1/zbcbt2gc3augz

調べ方

ID、ワインの種類、味わいからワインを検索します。

Method: GET

例：フルボディの赤ワインの検索:https://sheetdb.io/api/v1/zbcbt2gc3augz/search?category=赤&body=フル

パラメーター

  • id: id (1~)

  • category: 赤, 白, スパークリング, ロゼ, オレンジ

  • body: フル, ミディアム＋, ミディアム, ミディアムー, ライト

  • taste: 辛口, やや辛口, やや甘口, 甘口
